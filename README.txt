# TwitchClipper
Project that fetches top twitch clips from a specific channel and puts them into an easy to upload mp4 file.

#STEPS FOR RUNNING
1. Install FFMPEG ver. ffmpeg version 2022-08-31-git-319e8a49b5-full_build-www.gyan.dev Copyright (c) 2000-2022 the FFmpeg developers
Download at: https://github.com/GyanD/codexffmpeg/releases/tag/2022-08-31-git-319e8a49b5

1. Make sure you have the following
    a.FFMPEG ver. ffmpeg version 2022-08-31-git-319e8a49b5-full_build-www.gyan.dev Copyright (c) 2000-2022 the FFmpeg developers
      Download at: https://github.com/GyanD/codexffmpeg/releases/tag/2022-08-31-git-319e8a49b5
    b.Python
    c.Your twitch app_id and app_secret. If you don't have it, look up how to do it. I did it using twitchAPI python library.

2. Make a virtual environment. Activate virtual environment.

3. install the required packages from requirements.txt with pip install -r requirements.txt

4. Run main.py
